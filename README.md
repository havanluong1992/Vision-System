# OT Tracker - Ứng Dụng Chấm Công Tăng Ca

Ứng dụng web đơn giản, hiện đại giúp ghi nhận và tính toán giờ tăng ca (OT) tự động.

## Tính Năng
- **Giao Diện Đẹp**: Thiết kế Dark Mode hiện đại, hiệu ứng Glassmorphism.
- **Tự Động Tính**: Nhập giờ về, app tự trừ giờ chuẩn (mặc định 17:30) để ra giờ OT.
- **Lưu Trữ**: Dữ liệu được lưu ngay trên trình duyệt (không sợ mất khi tải lại trang).
- **Lịch Sử**: Xem lại lịch sử chấm công theo tháng.
- **Cài Đặt**: Tùy chỉnh giờ tan ca tiêu chuẩn.

## Cách Sử Dụng (Dành cho Điện Thoại - PWA)
Vì bạn muốn dùng như một App trên điện thoại:

1. **Hosting (Đưa lên mạng)**:
   - Cách đơn giản nhất là đưa bộ code này lên GitHub Pages, Vercel hoặc Netlify (miễn phí).
   - Sau đó truy cập link web trên điện thoại.

2. **Cài Đặt vào Máy**:
   - **iOS (Safari)**: Ấn nút **Chia sẻ** (Share) -> Chọn **Thêm vào MH chính** (Add to Home Screen).
   - **Android (Chrome)**: Ấn menu **More** (3 chấm) -> Chọn **Cài đặt ứng dụng** hoặc **Thêm vào màn hình chính**.

## Cách Sử Dụng (Trên Máy Tính)
1. Mở file `index.html` bằng trình duyệt (Chrome, Edge, Cốc Cốc).
2. Sử dụng bình thường.

## Cấu Trúc Thư Mục
- `index.html`: Giao diện chính.
- `style.css`: Các đoạn mã trang trí, màu sắc.
- `script.js`: Xử lý tính toán và lưu dữ liệu.
- `manifest.json`: File cấu hình để cài đặt như App.
